ข้อมูลหลักสูตร

ชื่อหลักสูตร: วิวิศวกรรมศาสตรบัณฑิต (วิศวกรรมสารสนเทศและเครือข่าย)

รหัสหลักสูตร: 67062034

ระดับการศึกษา: ปริญญาตรี

คณะ / มหาวิทยาลัย: คณะเทคโรโลยีและการจัดการอุตสาหกรรม มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ วิทยาเขตปราจีนบุรี

คำอธิบายหลักสูตร: INE คือหลักสูตรวิศวกรรมที่เรียนเกี่ยวกับระบบเครือข่าย สารสนเทศ และเทคโนโลยีดิจิทัล เช่น IoT และความปลอดภัยไซเบอร์
